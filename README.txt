woocommerce_thai
================
วิธีนำไปใช้ครับ

ผมได้เปลี่ยนเครื่องมือในการแปลมาใช้ Loco Translate ทำให้เอาไปใช้ด้เลยไม่ต้องรีเนมเหมือนรุ่นก่อนๆแล้วครับ

1. ดาวน์โหลดไฟล์ woocommerce-th.mo และ woocommerce-th.po ที่นี่ครับ https://github.com/taninbkk/woocommerce_thai/archive/master.zip
2. นำทั้งสองไฟล์ไปไว้ในโฟล์เดอร์ /public_html/wp-content/languages/plugins

** ถ้าอับเดท woocommerce แล้วไฟล์หาย กลับมาโหลดใหม่แล้วทำตามขั้นตอนเดิมครับ
** ไฟล์ที่แปลยังไม่สมบูรณ์นะครับ ผมมีเวลาเมื่อไหร่จะแปลให้ครบครับ
